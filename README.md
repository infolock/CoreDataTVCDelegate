CoreDataTVCDelegate
===================

![Unmaintained](https://img.shields.io/badge/%E2%9A%A0-unmaintained-red.svg?style=flat)

This is just a simple object to be used with UIViewControllers that have a UITableView.  Simply extend this object rather than UIViewController for usage.

Be sure to implement -(void)configureCell:(UITableViewCell *)cell atIndexPath:(NSIndexPath *)indexPath when extending.
